# Code of Conduct

## Our Commitment

In the interest of fostering an open, respectful, and professional environment, we as contributors and maintainers pledge to make participation in the BlockScope project a harassment-free experience for everyone.

BlockScope is an academic and collaborative software project. Professional conduct is expected at all times.

---

## Expected Behavior

All contributors are expected to:

- Be respectful and constructive in discussions
- Accept feedback professionally
- Focus on technical issues, not individuals
- Provide helpful and actionable code reviews
- Communicate clearly in pull requests and issues
- Maintain academic integrity

---

## Unacceptable Behavior

The following behaviors are not tolerated:

- Harassment, discrimination, or personal attacks
- Public or private abusive language
- Dismissive or hostile code review comments
- Sharing confidential information
- Intentionally submitting malicious code
- Plagiarism or misrepresentation of work

---

## Contribution Integrity

All contributions must:

- Be original work or properly attributed
- Avoid copying external proprietary code
- Not include hardcoded secrets or credentials
- Follow project security standards

---

## Review Conduct

During code reviews:

- Critique the code, not the contributor
- Provide reasoning for requested changes
- Avoid vague comments like “This is bad”
- Suggest improvements when possible

---

## Reporting Issues

If you experience or witness unacceptable behavior:

1. Report it privately to the project maintainer.
2. Provide clear details of the incident.
3. Maintain confidentiality.

All reports will be reviewed and handled appropriately.

---

## Enforcement

Project maintainers reserve the right to:

- Request changes in behavior
- Remove inappropriate comments
- Reject pull requests
- Restrict contributor access if necessary

---

## Scope

This Code of Conduct applies to:

- GitHub discussions
- Pull requests
- Issues
- Project-related communication channels

---

By contributing to BlockScope, you agree to abide by this Code of Conduct.
